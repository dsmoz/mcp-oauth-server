Typography

Avenir Next is the primary typeface — confirmed present on this machine. It is a geometric sans-serif that matches the DS-MOZ brand values of modernity, clarity, and confidence. Helvetica Neue serves as the fallback for environments where Avenir Next is unavailable. Raleway (DS-MOZ brand primary) requires a CDN and is not suitable for offline-first web — Avenir Next is the locally-available equivalent.

Icon Usage Rules

  • Use Light variant (1.5px stroke) throughout — never Bold or Fill
  • Size: 24px for nav/section headers · 20px for table actions · 16px inside buttons
  • Colour: always inherit from CSS color property — never hardcode in SVG fill attribute
  • Never mix Phosphor with other icon sets on the same page
  • No representational icons for organisations or populations — use abstract symbols only
  • Telegram-specific: use PaperPlaneTilt for notification send, ChatTeardropDots for bot interaction

Photography & Illustration

Visual guidance for imagery used in marketing materials, landing pages, and documentation for DSMOZ Intelligence.

Appropriate Imagery

  • Abstract technology — circuit patterns, data flows, network nodes — teal/dark-toned
  • Professional workspace: laptops, terminals, minimal desk setups — no clutter
  • Mozambique/Southern Africa context: authentic urban professional settings when human subjects are shown
  • Security metaphors: locks, keys, shields — rendered abstractly, not cartoonishly
  • Dark-mode screenshots of the product itself — most powerful asset
  • Composition: wide cinematic crops (16:9), subject off-centre, generous negative space

Avoid

  • Stock photo clichés: handshakes, people pointing at whiteboards, staged "diversity" photos
  • Bright white backgrounds — breaks the dark-mode product aesthetic
  • Blue/purple colour palettes from competing SaaS brands (Stripe, Linear, Vercel tones)
  • Generic "AI" imagery: glowing brain outlines, robotic hands, blue particle fields
  • Imagery that positions Africa/Mozambique as underdeveloped or aid-receiving
  • Faces without consent documentation

AI Illustration Rules

Use DS-MOZ SaaS Vector style (Recraft style ID: d353d61e) for product marketing assets. Colours in prompts: describe as "dark teal background, warm orange accents, clean white lines" — never paste hex codes.

  • Use for: hero section backgrounds, feature illustrations, onboarding empty states
  • Do not generate: realistic photographs of people, logos of any organisation, UI mockups (do these in HTML/CSS)
  • Output directory: ~/Desktop/design-engine-output/dsmoz-intelligence/
  • Always upload generated assets to Cloudinary (account: dq2ajrfxl) before use

Logo Usage on Dark Backgrounds

Use DSMC-White.svg on all dark surfaces (the product's native environment). Cloudinary URL: https://res.cloudinary.com/dq2ajrfxl/image/upload/v1742962914/white_dt1cfp.png
