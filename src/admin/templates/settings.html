Setup instructions

  1. Open Telegram, message @BotFather and send /newbot. Choose a name and username, then copy the HTTP API token into Bot Token below.

  5. Save settings, then redeploy or restart the server — webhook is registered on startup against OAUTH_ISSUER_URL/telegram/webhook.
