Claude Desktop

Open Claude Desktop → Settings → Connectors → Add custom connector. Paste:

  1. 1Name: DS-MOZ Connect (or any label)
  2. 2Remote MCP Server URL → the Gateway URL above
  3. 3Under Advanced: paste OAuth Client ID and Secret
  4. 4Click Add. Sign in when the browser opens. Tokens refresh automatically.

Prefer a config file? Download claude_desktop_config.json — uses mcp-remote to handle OAuth. Requires Node.js.

Claude.ai (Web)

Go to Claude.ai → Settings → Connectors → Add custom connector. Paste the Gateway URL, then expand Advanced for OAuth Client ID and Secret.

ChatGPT

Open Settings → Connectors → Create → New App (Developer mode required). Set Authentication to OAuth and paste the URL, Client ID, and Secret. Tick “I understand and want to continue”, then Create.

Claude Code (CLI)

Run once — Claude Code handles OAuth automatically. A browser opens for sign-in on first use; tokens refresh silently after.

claude mcp add --transport http --scope user gateway {{ gateway_me_url }}

The /gateway/me URL is shared — identity comes from your OAuth token.

Cursor

Cursor uses the same Streamable HTTP transport. Add to ~/.cursor/mcp.json:

{ "mcpServers": { "dsmoz": { "url": "{{ gateway_url }}" } } }

First request triggers OAuth in your browser.
