Claude Code (CLI)

Run once — Claude Code handles OAuth automatically. A browser opens for sign-in on first use; tokens refresh silently after that.

claude mcp add --transport http --scope user gateway {{ gateway_me_url }}

The /gateway/me URL is the same for all users — your identity is resolved from the OAuth token.
